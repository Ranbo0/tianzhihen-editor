# 天之痕 mon.dat 编辑器

一个基于浏览器的轩辕剑外传：天之痕（DOS 版）游戏数据二进制编辑器。无需安装，打开网页即可使用。

**在线使用：** https://ranbo0.github.io/tianzhihen-editor/

## 功能概览

- **活物编辑**：修改怪物/NPC 的等级、攻击、防御、敏捷、智慧、生命、经验、金钱、掉落物品、偷取物品等属性
- **消耗品编辑**：修改恢复道具、属性增加物品、增益/减益物品的效果值和目标
- **武器/防具编辑**：修改攻击、防御、敏捷、额外伤害、特效等属性，支持五行抗性编辑
- **法宝编辑**：修改体力消耗、装备者、熟练度
- **技能/奇术编辑**：修改灵力/体力/气消耗值、回复效果值，显示作用对象和效果类型
- **maps.dat 描述编辑**：加载 maps.dat 后可直接修改每个物品的文字说明
- **level.dat 升级表**：查看和修改四位主角每级的属性成长、习得奇术
- **目录加载**：一次性加载整个游戏目录，自动识别 mon.dat、maps.dat、level.dat
- **直接保存**：通过 File System Access API 直接覆写原文件（Chrome/Edge）

## 使用方法

### 在线使用

直接访问 https://ranbo0.github.io/tianzhihen-editor/ ，推荐使用 Chrome 或 Edge 浏览器。

### 加载文件

- 点击「加载目录」选择游戏所在文件夹，自动识别并加载所有 .dat 文件（推荐）
- 或分别点击「加载 mon.dat」「加载 maps.dat」「加载 level.dat」单独加载

### 编辑数据

- 表格中带输入框的字段可直接修改，修改后边框变为橙色表示已暂存
- 点击行可打开详细编辑弹窗
- 使用筛选器按类别查看：活物、恢复道具、武器、防具、法宝、技能等
- 支持按编号、名称、等级、代码排序，可切换升序/降序
- 每页显示条目数可选：25/50/100/200

### 保存修改

- 点击「保存 mon.dat」/「保存 maps.dat」/「保存 level.dat」写回文件
- 使用 File System Access API 时直接覆写原文件
- 不支持 API 的浏览器会触发下载

## 浏览器要求

- **推荐：** Chrome 86+ / Edge 86+（支持 File System Access API，可直接读写文件）
- Firefox / Safari 可用，但需通过上传/下载方式操作

## 支持的文件

| 文件      | 说明                                             |
| --------- | ------------------------------------------------ |
| mon.dat   | 主数据文件，包含活物、物品、装备、技能等所有条目 |
| maps.dat  | 物品描述文本（Big5 编码）                        |
| level.dat | 角色升级属性成长表                               |

## 数据类别

| 代码范围  | 类别             |
| --------- | ---------------- |
| 1-500     | 活物（怪物/NPC） |
| 3001-3024 | 恢复道具         |
| 3025-3034 | 解除状态         |
| 3035-3044 | 属性增加（上限） |
| 3045-3047 | 特殊物品         |
| 3048-3072 | 增益/减益        |
| 3151-3184 | 法宝             |
| 3201-3256 | 武器             |
| 3257-3267 | 暗器（投掷）     |
| 3271-3398 | 防具/饰品        |
| 6001+     | 技能/奇术        |

## 技术说明

- 单文件 HTML/JavaScript 应用，无外部依赖
- 使用 Big5 TextDecoder 解析中文名称和描述
- 支持暗色/亮色主题自动切换
